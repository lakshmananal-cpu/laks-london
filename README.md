# laks.london

Personal site for Laks — VP of Technology at Genpact. AI & technology leadership.

Static single-page site (`index.html`), no build step. Trading-terminal design with a live data-tape rail, JetBrains Mono / Space Grotesk / Inter type.

## Deploy
- **GitHub Pages:** Settings → Pages → deploy from `main` / root.
- **Vercel:** import the repo; framework preset "Other"; no build command; output dir `/`.

Custom domain `laks.london` configured via the `CNAME` file (GitHub Pages) or Vercel's domain settings.
